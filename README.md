## 🔭 Circuit Design Sandbox

### 1. Single & Two-Qubit Foundations
* **Pauli Rotations:** Implementing pauli X,Y, and Z gates for state manipulation.
* **Entanglement:** Constructing Bell States using Hadamard ($H$) and $CNOT$ configurations.

### 2. Reversible Logic Gates
* **Toffoli (CCNOT):** Building 3-qubit controlled-NOT circuits for universal reversible computing.
* **Fredkin (CSWAP):** Designing controlled-swap networks for quantum data routing.

### 3. Algorithmic Circuit Architectures
* **Deutsch & Deutsch-Jozsa:** Mapping quantum oracles ($U_f$) to evaluate function properties in a single run.
* **Grover's Search:** Constructing phase diffusers and oracle structures for amplitude amplification.
* **Shor's Algorithm:** Exploring the circuit layout for Quantum Fourier Transforms (QFT) and modular exponentiation.
